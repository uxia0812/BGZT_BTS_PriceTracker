## 데이터 분석 인사이트

### 통계
- **총 거래 건수**: 50,000+
- **분석된 포토카드**: 299종류
- **멤버별 분포**: 7명 + 단체
- **앨범별 분포**: 15개 주요 앨범
- **특수 타입**: 10가지 (럭드포, 위버스포, 공포 등)

### 시세 분석
- **중앙값 기준**: 이상치 제거 (IQR 방법)
- **최소 거래 수**: 3건 이상만 포함
- **시계열 데이터**: 최근 30개 거래 추이
- **가격 범위**: 최저/최고/평균 표시

## 기술적 성과

### 데이터 처리 파이프라인
```
Redash API (글로벌번장 DB)
    ↓
Python 분석 스크립트
    ↓
데이터 정규화 (멤버/앨범/타입)
    ↓
시세 계산 (중앙값, 이상치 제거)
    ↓
HTML 생성 (Chart.js 그래프)
    ↓
Vercel 배포
```

### 자동화 가능성
- 매일/매주 자동 데이터 업데이트
- GitHub Actions 또는 cron job
- 실시간 시세 반영 가능
